Project Axeman is JavaScript extension for Google Chrome that will inject custom script to Travian browser game page. Those custom scripts will add additional functionality to the game. It is customizable and easy to use. This means that you can enable or disable each functionality to get better game experience.

This extension works using plugins. There are may plugins available to customize your game. You can easily enable or diasble individual plugins by going into Plugins Manager and click on On/Off button under each plugin's image. Those plugins put additional functionality in to the game. They inject custom scripts and can change page layout so that they help you be the best in the game.

Download

You can get extension from Chrome Web Store. When you install extension via store, you will recieve all available updates. This is best for most users.

Advanced

Users that want to change script to fit theyr needs have other option. Download source code and load it to Chrome directly. You can download source code in either zip or tar formats. Caution! You will not recieve any updates if you directly load source code. In order to recieve latest updates, install extension via Chrome Web Store.

If you have great idea or you just want to help, you can also clone the project with Git by running: 

$ git clone git://github.com/JustBuild/Project-Axeman

Plugins are scripts that are independent. They can be enabled or disabled without making any changes other scripts. This is nice feature to users and developers. Developers can now add functionality without knowing rest of the extension in details.
